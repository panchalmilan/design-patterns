### Stock Trading App


- Publishers _(NSE, BSE)_ will constantly publish data _(stock name, price, version)_
  - e.g.
    - NSE: AMZ, 100, 1
    - BSE: AMZ, 101, 2
  - here BSE sends latest data of AMZ stock as it has higher version than NSE
- Subscribers can connect to multiple subscriber to receive updates
